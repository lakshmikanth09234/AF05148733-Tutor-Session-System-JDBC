# Tutor Session Scheduling System (Java JDBC)

This is a console-based application to schedule tutor sessions based on subject and time slots using Java and JDBC.

## 🚀 Features
- User registration and login
- View tutors by subject
- Schedule sessions with time slots
- Cancel sessions
- View user-specific sessions

## 🛠 Technologies Used
- Java
- JDBC
- MySQL

## 🧠 Concepts Used
- Authentication system
- Session management
- Filtering data using SQL
- Structured application flow

## ▶️ How to Run
1. Create database and tables in MySQL
2. Update DB credentials
3. Run Main.java

## 📌 Highlights
- Subject-based tutor filtering
- Time-slot scheduling system
- User-specific data handling
